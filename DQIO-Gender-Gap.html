Guided Project: Visualisating the Gender Gap in College Degrees [US]¶

This notebook is the 5th guided project from datacamp.io and it works on a csv dataset that holds the % of women obtaining degree's across certain categories/studies. We are able to therefore work out the men % by using this dataset.

The aim of this notebook is to put into practice some learned concepts about reducing chartjunk and improving the data-ink ratio for the viewer to quickly gain a highlevel overview.

It also practices running multiple charts in a "column" format, showing data from 3 categories.

# A quick set of plots to show what the data looks like and how we can work out the men's values
%matplotlib inline
import pandas as pd
import matplotlib.pyplot as plt

women_degrees = pd.read_csv('percent-bachelors-degrees-women-usa.csv')
cb_dark_blue = (0/255,107/255,164/255)
cb_orange = (255/255, 128/255, 14/255)
stem_cats = ['Engineering', 'Computer Science', 'Psychology', 'Biology', 'Physical Sciences', 'Math and Statistics']

fig = plt.figure(figsize=(18, 3))

for sp in range(0,6):
    ax = fig.add_subplot(1,6,sp+1)
    ax.plot(women_degrees['Year'], women_degrees[stem_cats[sp]], c=cb_dark_blue, label='Women', linewidth=3)
    ax.plot(women_degrees['Year'], 100-women_degrees[stem_cats[sp]], c=cb_orange, label='Men', linewidth=3)
    ax.spines["right"].set_visible(False)    
    ax.spines["left"].set_visible(False)
    ax.spines["top"].set_visible(False)    
    ax.spines["bottom"].set_visible(False)
    ax.set_xlim(1968, 2011)
    ax.set_ylim(0,100)
    ax.set_title(stem_cats[sp])
    ax.tick_params(bottom="off", top="off", left="off", right="off")
    
    if sp == 0:
        ax.text(2005, 87, 'Men')
        ax.text(2002, 8, 'Women')
    elif sp == 5:
        ax.text(2005, 62, 'Men')
        ax.text(2001, 35, 'Women')
plt.show()

Letting it all hang out¶

Lets produce a chart for each of the degree subjects. To do this we will put them into categories, Stem, librial arts and other, and will then display each category in a single column, multi-row layout of grids. 6x3, ordering them in decending order of women % obtaining the degree.

fig = plt.figure(figsize=(16, 20))

# 1st for loop to produce the first column of charts
# Jump 3 numbers from 0 to 18. So 0,3,6,9,12,15,18. This will place
# the graphs from this loop into the first column of a 3 column figure.
# We go to 18 in the sp range even though the last plot takes up space 15
# this is because the loop does not include 18, it stops at 18, with a jump
# of 3, the last subplot-space produced is 15.
for sp in range(0,18,3):
    list_index = int(sp/3) # create index number for stem_cat list
                            # 0/3=0, 3/3=1, 6/3=2 9/3=3,

    ax = fig.add_subplot(6,3,sp+1) # 6 rows, 3 column figure. Places start from 1 not zero.
    ax.plot(women_degrees['Year'], women_degrees[stem_cats[list_index]], c=cb_dark_blue, label='Women', linewidth=3)
    ax.plot(women_degrees['Year'], 100-women_degrees[stem_cats[list_index]], c=cb_orange, label='Men', linewidth=3)
    
    for spine in ax.spines:
        ax.spines[spine].set_visible(False)
    
    ax.axhline(50, c=(171/255, 171/255, 171/255), alpha=0.3)
    ax.set_xlim(1968, 2011)
    ax.set_ylim(0,100)
    ax.set_yticks([0, 100])
    ax.set_title(stem_cats[list_index])
    ax.tick_params(labelbottom="off", bottom="off", top="off", left="off", right="off")

    if sp == 0: # first object in this column
        ax.text(2005, 87, 'Men')
        ax.text(2002, 8, 'Women')
    elif sp == 15: # Last object in this column
        ax.text(2005, 62, 'Men')
        ax.text(2001, 35, 'Women')
        ax.tick_params(labelbottom="on")
        
# 2nd for loop to produce the 2nd column of charts
# Jump 3 numbers from 1 to 16 (there are only 5 values in the list).
# to place charts in positions 1,4,7,10,13,16
# subplot-space 16 will never be used (highend of range is a stop point, not inclusive)
for sp in range(1,16,3):
    list_index = int((sp-1)/3)
    # Index values to the list (1-1)/3=0, (4-1)/3=1, (7-1)/3=2, (10-1)/3 = 3...(13-1)/3=4

    ax = fig.add_subplot(6, 3, sp+1) # 6 rows, 3 column figure. Places start from 1 not zero.
    ax.plot(women_degrees['Year'], women_degrees[lib_art_cats[list_index]], c=cb_dark_blue, label='Women', linewidth=3)
    ax.plot(women_degrees['Year'], 100-women_degrees[lib_art_cats[list_index]], c=cb_orange, label='Men', linewidth=3)
    
    for key, spine in ax.spines.items():
        spine.set_visible(False)
    
    ax.axhline(50, c=(171/255, 171/255, 171/255), alpha=0.3)
    ax.set_xlim(1968, 2011)
    ax.set_ylim(0, 100)
    ax.set_yticks([0, 100])
    ax.set_title(lib_art_cats[list_index])
    ax.tick_params(labelbottom="off", bottom="off", top="off", left="off", right="off")

    if sp == 1: # first object in this column
        ax.text(2005, 87, 'Men')
        ax.text(2002, 8, 'Women')
    elif sp == 13: # Last object in this column
        ax.text(2005, 62, 'Men')
        ax.text(2001, 35, 'Women')
        ax.tick_params(labelbottom="on")

for sp in range(2,20,3):
    list_index = int((sp-2)/3)
