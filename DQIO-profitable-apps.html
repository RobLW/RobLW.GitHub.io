Revenue for apps in Apple App Store and Google Play¶

This notebook demonstrates the profitability of in-app advertisement for free-to-download applications on Google Play and the App Store. The data source is taken from Kaggle. This is the first guided project presented on Dataquest.io, and so the aim here is not to produce something that is efficient, but to build a foundational knowledge on using Python for data analytics. This means rather than using pandas it'll be all driven by lists and loops where necessary.

This notebook is working on a sample of the data that is made available via Kaggle, as it is for a learning experience and faster performance is key to a fluid learning process.

  • https://www.kaggle.com/lava18/google-play-store-apps/home
  • https://www.kaggle.com/ramamet4/app-store-apple-data-set-10k-apps/home

def engApp(appname):
    counter = 0
    for char in appname:
        if ord(char) > 127:
            counter += 1
    if counter > 3:
        return False
    else:
        return True

# Function to allow user to pass a list of lists as dataset, and column number 
# to the index value. function loops through lists and pulls out the values
# based on the index, to put into a new local dictionary and returns it.

def freq_table(dataset, index):
    freq = {}
    for each in dataset:
        if each[index] in freq:
            freq[each[index]] += 1
        else:
            freq[each[index]] = 1
    
    return freq

def freq_table_percentage(dataset, index):
    freq = {}
    for each in dataset:
        if each[index] in freq:
            freq[each[index]] += 1
        else:
            freq[each[index]] = 1
    
    total = sum(freq.values())
    for k, v in freq.items():
        freq[k] = round((v/total) * 100,2)
    
    return freq

def display_table(dataset, index):
    table = freq_table_percentage(dataset, index)
    table_display = []
    for key in table:
        key_val_as_tuple = (table[key], key)
        table_display.append(key_val_as_tuple)
        
    table_sorted = sorted(table_display, reverse = True)
    for entry in table_sorted:
        print(entry[1], ':', entry[0])
